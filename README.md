## General info

FrontEnd App which shows current status of covid in specific caountry on map.

## Technologies

Project is created with:

- JavaScript
- React
- HTML
- react-chartjs-2
- axios
- react-google-charts

## Setup

1. Download
2. run cmd
3. Go to directory
4. npm install
5. npm start
6. run browser on localhost:3000
